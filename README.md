# 语雀小记导出助手 (Yuque Note Exporter)

**语雀小记导出助手** 是一个轻量级、高性能的 Chrome 浏览器扩展。它旨在解决语雀小记（Notes）“只记不看”、“导出困难”的痛点，帮助你将碎片化的灵感一键导出为纯净的 Markdown 格式，是 **投喂 AI (ChatGPT/Claude/DeepSeek)** 进行深度复盘的绝佳伴侣。

## ✨ 核心亮点

* **🎯 精准筛选**：告别大锅烩！支持按**“标签”**（多选）和**“日期范围”**（最近一周/自定义）组合筛选。
* **🤖 AI 友好型导出**：导出的内容自动清洗 HTML 标签，生成最干净的 Markdown 格式，带有清晰的时间戳和标签元数据，AI 读起来毫无障碍。
* **🔒 数据安全**：所有请求直接与语雀官方接口通信，数据仅存储在本地浏览器，不经过任何第三方服务器。

## 🚀 最佳实践：如何用它做 AI 复盘？

这个插件最强大的用法，是作为你和 AI 之间的桥梁：

1.  **筛选**：在插件中选择 `最近一周` 的时间范围。
2.  **导出**：点击导出，获得一个 `.txt` 文件。
3.  **投喂**：将文件发送给 AI（如 ChatGPT），并发送以下提示词（Prompt）：

> "这是我过去一周的碎片想法记录。请帮我分析：
> 1. 我这周的情绪主要处于什么状态？
> 2. 我关注最多的 3 个关键词是什么？
> 3. 有哪些未完成的灵感或待办事项？"

你会发现，你随手记下的碎片，竟然藏着这么多关于你潜意识的秘密。

## 🛠 安装教程

由于目前插件尚未上架 Chrome 应用商店，你可以通过“开发者模式”免费安装：

1.  **下载代码**：点击本页面右上角的 `Code` -> `Download ZIP`，下载并解压到本地文件夹。
2.  **打开扩展管理**：在 Chrome 浏览器地址栏输入 `chrome://extensions/` 并回车。
3.  **开启开发者模式**：打开右上角的 `开发者模式` 开关。
4.  **加载插件**：点击左上角的 `加载已解压的扩展程序`，选择你刚才解压的文件夹。
5.  **开始使用**：在浏览器右上角找到🧩拼图图标，将“语雀小记导出助手”固定到工具栏。

## 📖 使用说明

1.  确保你已经**登录**了 [语雀网页版](https://www.yuque.com/)。
2.  点击插件图标，它会自动同步你所有的标签。
3.  选择你想导出的 **标签**（如“日记”、“灵感”）或 **时间段**。
4.  点击 **“搜索”** 查看条数，然后点击 **“导出”** 下载文件。
